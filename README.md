# Superstore Sales Performance Dashboard

## Project Overview
This project analyzes Superstore sales data using Power BI to identify sales trends, category performance, regional sales, and top products.

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization

## Dashboard Features
- Total Sales
- Total Profit
- Total Quantity Sold
- Sales Trend Analysis
- Sales by Category
- Top Products by Sales
- Sales by Region
- Profit by Region
- Region Filter
- Category Filter

## Key Findings
- Total Sales reached $2.30M.
- Total Profit reached $286.40K.
- Total Quantity Sold was 38K units.
- Technology category generated the highest sales.
- West region recorded the highest sales and profit.
- Sales showed steady growth from 2014 to 2017.

## Recommendations
- Focus marketing efforts on Technology products.
- Increase inventory for top-selling products.
- Expand operations in West and East regions.
- Improve strategies in Central and South regions.
